# Функция для проверки модов

Функция `"SomeModOn()"`, возвращает либо `"да"`, либо `"нет"` `(истина/ложь, true/false, 0/1)`. Принимает один опциональный аргумент, по умолчанию — "имя_предмета_из_мода".

Внутри проходится по названиям всех акторов из всех подключённых модификаций. Если какое-нибудь совпадает с целевым — возвращаем "true" (найдено). 
Если после прохода по всему массиву названий ничего подходящего не нашлось — говорим, что false, мол, такого актора среди подключённых нет.

```C
bool SomeModOn( String testcls = "имя_предмета_из_мода" ) {
  for ( int i = 0; i < AllActorClasses.Size(); i++ )
      if ( AllActorClasses[ i ] == testcls )
          return true;

    return false;
}
```
